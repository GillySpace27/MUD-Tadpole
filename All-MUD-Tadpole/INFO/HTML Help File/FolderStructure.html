Folder Structure

  • TADPOLEGIT
    • Contains Everything



  • All-MUD-Tadpole
    • This has all you need to run the program
  • Archive Functions
    • Lots of old and mostly deprecated things that have been done by either Pam, Jake, or Chris/Conor. May contain clues.



  • SRC
    • MUD_MAIN
      • The primary function that runs the experiment.
  • LIB
    • Math
      • A plethora of basic functions required to operate on the data. A lot of this has to do with transforming between domains.
    • Primary Functions
      • Each of the steps along the workflow that perform major operations on the data.
    • Thorcam + ESP Stage
      • The functions that directly interface with and control the physical experiment.
  • INFO
    • Paper Describing the Temporal Interleaving Algorithm
    • Help Documentation
    • Graphs
