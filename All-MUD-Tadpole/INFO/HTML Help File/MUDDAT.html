MUDDAT

This file represents all of the information provided by the experiment, be it simulated or real. It contains the raw data from each cross section, and has been analyzed to extract the spectrum and the spectral phase.


This is 1xN structure array, where N is the number of cross section traces taken. It contains the following fields in common with MUDPIX:

  • Data
    • This is the raw cross section
  • lam
    • This is a vector labeling the wavelength axis
  • x
    • This is a vector labeling the spacial axis
  • pos
    • This is a scalar value, the position of the delay stage in mm



In addition to the following new fields:


  • spec
    • The spectrum of the trace, with respect to lam
  • rawphi
    • The spectral phase difference between the reference pulse and the cross section of the unknown pulse. (For Debug Purposes)
  • phi
    • The spectral phase of the trace, with respect to lam
  • success
    • 1 - if the algorithm was able to extract the spectrum/phase of this trace
    • 0 - if the algorithm failed to extract the spectrum/phase of this trace
