PULSE


This object contains all of the information about a pulse. It is the final deliverable of the program.


PULSE has the following fields:

  • Time Domain
    • t
      • The vector labeling the time axis
    • w
      • The vector labeling the w axis
    • int
      • The temporal intensity of the pulse
    • tphi
      • The temporal phase of the pulse


  • Wavelength Domain
    • lam
      • The vector labeling the lam axis
    • spec
      • The spectrum of the pulse
    • sphi
      • The spectral phase of the pulse
