SEA TADPOLE


Conceptually, SEA Tadpole is very straightforward. We take two temporally coincident pulses and we spectrally disperse them with a diffraction grating in the x direction. The pulses overlap to create spectral fringes in the x direction. Because the pulses emerge from the fiber conically, the beams also cross each other vertically at an angle, creating spacial fringes in the y direction. This gives us traces that look like the following:



To the left and right of the trace you can see the retrieved spectrum and spectral phase of the pulse. (Underneath the hood, the algorithm is just returning the phase difference between the reference trace and the unknown trace. We must then add the reference pulse's spectral phase as determined by a GRENOULLIE to find the true phase)


This technique is great, as long as the unknown pulse is shorter than the reference pulse, and they both contain all the same colors. But what if you want to measure a long, very complex pulse?
