MUD_MAIN


First, the MUDPIX and REF objects are either captured or simulated.


These are then passed to extract_MUDPIX, which generates the MUDDAT object. This is the program that analyzes the raw traces and extracts the usable data from them.


If there is only a single trace, MUDDAT is given to sea_extract_MUDDAT, which returns the PULSE.


If there are two or more traces, MUDDAT is passed to temporal_interleave, which returns the PULSE.
